# 🏨 Hotel Reservation System

This is a **Java-based Hotel Reservation System** built as part of the CodeAlpha Internship. The system allows users to view available rooms, book reservations, simulate payments, cancel bookings, and store data persistently using file handling — all through a clean console-based interface.

---

## 🔧 Features

- 🛏️ View available hotel rooms
- 📋 Book a room by entering customer details
- 💳 Simulated payment with dummy card input
- ❌ Cancel a reservation by name and room number
- 🧾 View all existing bookings
- 💾 Persistent data storage using `rooms.txt` and `reservations.txt`
- 🧠 Built with clean OOP concepts (`Room`, `Customer`, `Reservation`, `BookingManager`)
- ☕ Java Console Application
